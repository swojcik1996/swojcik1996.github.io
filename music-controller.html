Music Controller

By Sławomir Wójcik Dec 08, 2017 Project

Music Controller it is a desktop application written in Java using JavaFX allowing you to add sound effects to your music. It has a registration and login system using the OrmLite database. After login in, the keys on the keyboard are used to play the sound by holding the button down. The number keys have the function of looping music. On the left and right side there is a music player from whichg we play any song we want to remix. The video from the application below.
