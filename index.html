       Numeric Character Recogniction Learning is a application written for the bachelor's thesis. I used Java, Maven, and the Neurph library to create the program.The application is used to learn the recognition of numeric symbols using a neural network based on data from a file (learning data was generated using the "Generator Binary Digital" application).The result of running the application is the result of learning the neural network for each digit from the range 0-9. The table below shows the result of the mean of the 10 launches of the application with 43 neurons hidden layer, the results are given with an accuracy of 4 decimal places. Average total the result of recognizing 10 digits using the application is 90% (0.9074).
       The result was improved by increasing the number of neurons hidden from 43 to 2000 what resulted in achieving the result of 99% recognition of each digit. The result is an average from 10 application launches. The table below presents detailed results.        Generator Binary Digital it is a application having a graphical interface that was created using JavaFX technology and using the Scene Builder tool It is an application that generates the given data to the neural network (Numeric Character Recogniction Learning). The application screen shows a 7x12 matrix on which the user draws number. Clicking the left mouse button on a single matrix cell changes its color andvisually shows the current state of the drawn number symbol.
       When the application user draws a number in the matrix and writes the correct result in the field text, then click the save button, a record is added to the text file a teacher consisting of 94 numbers separated by commas. The first 84 numbers represent the input data, which is a reflection of the matrix 7x12 representing the symbol drawn, and subsequent digits are the expected output values obtained after conversion of the number given from the text field, meaning the correct result drawn number.The following graphic shows a text file containing a set of numbers between 0 and 9 created on the basis of the Generator Binary Digital application.
