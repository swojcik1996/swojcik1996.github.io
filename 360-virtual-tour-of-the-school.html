360 Virtual tour of the school

By Sławomir Wójcik Dec 08, 2017 Project

360 Virtual tour of the school it is a project created by a three-person team. We used technologies such as Java Script and JSON. It allows users to visit the entire building and out-of-school areas. In addition, the project has Debug option which allows developers to easily read points between panoramas.
