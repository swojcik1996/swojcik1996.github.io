Kontroler muzyki

Sławomir Wójcik Grudzień 08, 2017 Projekt

Music Controller is a desktop application written in Java using JavaFX allowing you to add sound effects to your music. It has a registration and login system using the OrmLite database. After login in, the keys on the keyboard are used to play the sound by holding the button down. The number of keys has the function of looping music. On the left and right side, there is a music player from which we play any song we want to remix. The video from the application below.
